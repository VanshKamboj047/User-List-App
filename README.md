# User List App

A React Native app that fetches users from JSONPlaceholder API and displays them in a searchable list.

## Features
- Fetch users from API
- Search by name or email
- User detail screen
- Pull to refresh
- Loading and error handling

## Tech Stack
- React Native
- Expo
- React Navigation
- Axios

## Setup
1. Clone repo
2. Run npm install
3. Run npx expo start

## API
https://jsonplaceholder.typicode.com/users
